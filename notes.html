Drop-CapSome keywords or description

Drop-cap technique used in this template is experimental and may not be the best solution for it. It is easy to use. Just add the following to the first letter of your paragraph. <b class="drop-*"><span>*</span></b> and replace the * with the first letter of your paragraph.

If you don't like it, just delete remove the link to dropcap.css in the '<head>' and it will help to speed up the loading time.

Example usage : - if you use <b class="drop-a"><span>A</span> it will look like this :-
